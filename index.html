Important Notice!
It has been a long run and I hope that Elf Emulation has contributed usefully to the 1802 community, but with the increasing costs of web hosting I can no longer keep this site running. I am not sure when this site comes up for renewal, but when it does I will not be renewing this site. As such, if there is anything on this site you would like to have, please get it now. All of my 1802-based source code has been moved to GitHub and can be found at: https://github.com/rileym65. I thank you all for your support over the many years that I ran this site.
Mike Riley
