This is the prototype for my Pico/Elf project. This machine was designed as the smallest fully functional Elf class computer. The definition for fully functional was that it must run Elf/OS. This board has 32k of RAM, 32k ROM containing the Elf/OS install package and my BIOS, a Q/EF? (jumper selectable) serial port, the 8-bit IDE/CF interface, and Micro/Elf version 2 expansion bus.

This machine has circuitry to swap the upper and lower 32k memory banks at boot time. This makes it possible to boot from the EPROM. Once any address in the upper 32k address block is accessed the memory banks are swapped so that RAM is in the lower 32k and the ROM is in the upper 32k
