This board is a prototype color graphics board for the Micro/Elf. It is based upon the TMS9118 Video Display Processor from TI. This chip delivers 256x192 resolution in up to 16 colors. Video ram is seperate from the main processor ram and the VDP does not need to interrupt the 1802 at any time during video display.

This board also contains an interface to PS/2 keyboards.
